# 給与明細 自動送信スクリプト(Google Apps Script)

PDF編集アプリで「番号」を付けて分割・保存した給与明細PDFを、Googleドライブの指定フォルダに置くと、
番号をもとに名簿(スプレッドシート)からメールアドレスを検索し、Gmailで自動送信するスクリプトです。

## 全体の流れ

1. **PDF編集アプリ**(このリポジトリのWebアプリ)で、1ページに複数枚まとまった給与明細を分割し、
   それぞれに従業員番号を付けてZIPで保存する
2. ダウンロードしたZIPを解凍し、中のPDFファイルをGoogleドライブの「監視対象フォルダ」にアップロードする
3. **このApps Scriptを実行**すると、フォルダ内の各PDFのファイル名から番号を読み取り、
   スプレッドシートの名簿と照合してメールアドレスを特定し、Gmailで送信する
4. 送信済みのファイルは自動的に「送信済みフォルダ」に移動される(二重送信防止)

## セットアップ手順

### 1. 名簿スプレッドシートを作る

新しいGoogleスプレッドシートを作成し、1行目にヘッダーを入れます。

| 番号 | 氏名 | メールアドレス |
|---|---|---|
| 1042 | 山田 太郎 | yamada@example.com |
| 1078 | 佐藤 花子 | sato@example.com |

「番号」列は、PDF編集アプリで分割したときに付けた番号(ファイル名の末尾の数字)と
必ず一致させてください。

### 2. Googleドライブにフォルダを2つ作る

- 監視対象フォルダ(例:「給与明細_受信」)— ここにPDFをアップロードする
- 送信済みフォルダ(例:「給与明細_送信済み」)— 送信後、自動でここに移動される

それぞれのフォルダを開き、URLの末尾からフォルダIDを控えます。

```
https://drive.google.com/drive/folders/【この部分がフォルダID】
```

### 3. スクリプトを設置する

1. 手順1で作ったスプレッドシートのメニューから「拡張機能」→「Apps Script」を開く
2. デフォルトのコードを削除し、[`SendPayslips.gs`](./SendPayslips.gs) の内容を貼り付ける
3. ファイル先頭付近の `CONFIG` の中の `FOLDER_ID` と `SENT_FOLDER_ID` を、手順2で控えたIDに書き換える
4. 保存する(Ctrl+S)

### 4. テスト実行する

1. 関数選択のプルダウンで `sendPayslips` を選び、実行(▶)ボタンを押す
2. 初回は権限確認画面が出るので、内容を確認して「許可」する
3. 実行後、「表示」→「ログ」でログを確認する。`CONFIG.TEST_MODE` が `true` の間は
   実際には送信されず、「どのファイルが誰に送られる予定か」だけがログに表示される
4. 内容が正しければ、`CONFIG.TEST_MODE` を `false` に書き換えて再度実行する

### 5. 自動実行を設定する(任意)

毎回手動で実行しなくても良いように、時間主導型トリガーを設定できます。

1. 左側メニューの時計アイコン(トリガー)を開く
2. 「トリガーを追加」→ 実行する関数: `sendPayslips` → イベントのソース: 時間主導型
   → 分ベースのタイマー(例: 10分おき) を選んで保存

これで、監視対象フォルダに新しいPDFが置かれるたびに、自動的にメール送信されるようになります。

## 注意事項

- `GmailApp.sendEmail` は1日あたりの送信件数に上限があります(個人のGmailアカウントで約100通/日、
  Google Workspaceアカウントではプランに応じて多くなります)。大量送信する場合はご注意ください。
- ファイル名は変更せずそのまま添付されます。
- 番号が名簿に見つからなかったファイルや、ファイル名から番号を読み取れなかったファイルは、
  送信されずログにスキップ理由が記録されます。フォルダに残ったままになるので、名簿を修正してから
  再実行してください。
